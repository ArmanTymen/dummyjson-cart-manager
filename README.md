🛒 Cart Management System (Test Task)
SPA-приложение для управления корзинами пользователей, построенное на базе публичного API DummyJSON.

🛠 Стек технологий
React (Vite) — быстрый запуск и сборка.

TypeScript — строгая типизация для стабильности кода.

React Query (@tanstack/react-query) — управление серверным состоянием, кэширование и мутации.

Zustand — легкий глобальный стейт для хранения пагинации.

Emotion (@emotion/styled) — стилизация компонентов (CSS-in-JS).

React Router — навигация и динамические роуты.

🏗 Архитектура (FSD)
Проект построен по методологии Feature-Sliced Design. Это позволяет четко разделить ответственность между модулями и делает код легко поддерживаемым.

app/ — инициализация приложения, роутинг и провайдеры.

pages/ — композиция виджетов для формирования полноценных страниц.

widgets/ — крупные блоки (список корзин, детальная таблица).

features/ — интерактивные действия (удаление товара, изменение количества, пагинация).

entities/ — бизнес-сущности (API запросы, типы данных, хуки сущностей).

shared/ — переиспользуемые UI-компоненты, утилиты форматирования, конфигурация API.

⚙️ Особенности реализации
Optimistic / Manual UI Updates: Так как API DummyJSON не сохраняет изменения на сервере, в проекте реализовано ручное обновление кэша React Query (setQueryData). Это позволяет видеть изменения (удаление, пересчет сумм) мгновенно и сохранять их при навигации в рамках сессии.

Mobile-First Design: Весь интерфейс полностью адаптивен. На мобильных устройствах таблицы трансформируются в удобный карточный вид.

Performance: Использован React.lazy и Suspense для разделения кода (code-splitting), что ускоряет первую загрузку.

Stability: Реализован глобальный ErrorBoundary для предотвращения падения всего приложения при ошибках в ленивых модулях или рендеринге.

Reactive Filtering: Реализован быстрый поиск по userId. Благодаря интеграции Zustand и React Query, список обновляется мгновенно при вводе ID, с 
автоматическим сбросом пагинации для корректного отображения результатов.

📦 Установка и запуск
Клонировать репозиторий:

Bash

Установить зависимости:

Bash
npm install
Запустить проект:

Bash
npm run dev
